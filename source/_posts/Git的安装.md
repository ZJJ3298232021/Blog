# Git的下载
点击进入[Git官网](https://git-scm.com/)，然后点击download for windows
![Git](../img/Git1.png)
然后点击Windows64位的链接下载，下载速度可能有点慢，我这里有最新版的程序提供[下载](https://easylink.cc/8xc647)
下载完成后解压运行程序即可开始安装
如果下载连接失效的话你也可以通过二维码下载。
                                                                             ![二维码下载Git](../img/downloadGit.png)
