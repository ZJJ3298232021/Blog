# Git的下载
点击进入[Git官网](https://git-scm.com/)，然后点击download for windows
![Git](../img/Git1.png)
然后点击Windows64位的链接下载，下载速度可能有点慢，我这里有最新版的程序提供[下载](http://s39uged0n.bkt.clouddn.com/Git-2.42.0.2-64-bit.zip)
下载完成后解压运行程序即可开始安装
